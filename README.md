# sys-dorm-sys
学生宿舍管理系统
## 项目目标

+ 复习数据库相关知识：数据库设计、基本CRUD操作，关联查询操作，JDBC操作
+ 复习Servlet处理HTTP请求
+ 复习HTTP相关知识：请求方法、响应状态码、数据类型等
+ 学习项目开发流程

## 开发环境与技术栈

+ Windows/Mac/Linux
+ Maven
+ Servlet
+ MySQL
+ Jackson

## 项目功能

主要业务：管理学校宿舍信息，记录并管理学生住宿信息

+ 用户注册
+ 用户登录、会话管理
+ 用户信息管理
+ 宿舍楼管理
+ 寝室管理
+ 学生住宿管理（主要实现部分）
